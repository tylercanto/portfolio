Redes Corporativas • VPN • Active Directory • Segurança

Implementação de infraestrutura corporativa envolvendo integração entre unidades, VPN Site-to-Site, gerenciamento de usuários e controle centralizado das estações de trabalho.

◈ Visão Geral da Infraestrutura

O projeto consistiu na criação de uma infraestrutura de rede corporativa para integração entre duas unidades da empresa, permitindo comunicação segura entre os ambientes e centralização dos principais serviços utilizados pela operação.

A unidade localizada em Divinópolis precisava se comunicar com a unidade de Itaúna, permitindo que os colaboradores acessassem os sistemas corporativos de forma segura através da rede privada.

◈ Desafio Técnico

O principal desafio foi criar uma comunicação segura entre as unidades, garantindo autenticação dos usuários, controle de acesso e gerenciamento eficiente das estações utilizadas pelos colaboradores.

  • Integração da rede entre as unidades.
  • Criação de comunicação segura através de VPN.
  • Centralização dos sistemas principais na matriz.
  • Controle de acesso dos colaboradores.
  • Gerenciamento de mais de 70 computadores.

◈ Arquitetura de Rede Implementada

A infraestrutura foi estruturada utilizando firewall, comunicação VPN entre localidades e serviços centralizados para gerenciamento dos usuários e equipamentos.

  • pfSense: utilizado nas duas unidades para criação da VPN Site-to-Site, permitindo comunicação segura entre as redes corporativas.
  • Servidor Active Directory: implementado para gerenciamento centralizado de usuários, autenticação e controle dos acessos às máquinas.
  • Servidor da Matriz: responsável por centralizar os sistemas principais utilizados pela operação.
  • VPN Site-to-Site: responsável pela comunicação privada entre as unidades, garantindo que os serviços internos não ficassem expostos diretamente na internet.

◈ Gerenciamento de Usuários e Acessos

Foi implementado um modelo de controle baseado em usuários individuais, permitindo maior segurança e rastreabilidade dos acessos realizados pelos colaboradores.

  • Criação de usuários individuais para colaboradores.
  • Autenticação através do Active Directory.
  • Controle de acesso aos computadores corporativos.
  • Permissão de acesso condicionada à conexão com a rede VPN.

◈ Controle das Estações de Trabalho

O ambiente possuía mais de 70 máquinas utilizadas pelos colaboradores, sendo necessário implementar controle e gerenciamento da utilização dos equipamentos.

  • Monitoramento das estações de trabalho.
  • Controle de navegação dos usuários.
  • Bloqueio de redes sociais conforme política interna.
  • Gerenciamento da utilização dos computadores.
  • Aplicação de políticas de segurança.

◈ Gerenciamento com MakroLock

Foi utilizado o software MakroLock para gerenciamento das estações, permitindo maior controle sobre a utilização dos equipamentos e aplicação das regras definidas pela empresa.

  • Controle de acesso à internet.
  • Restrição de determinados conteúdos.
  • Bloqueio de redes sociais.
  • Acompanhamento da utilização das máquinas.

◈ Tecnologias Utilizadas

pfSense VPN Site-to-Site Active Directory Windows Server Firewall Mikrotik MakroLock Redes TCP/IP

◈ Resultado da Implementação

A implementação proporcionou uma comunicação segura entre as unidades, maior controle dos acessos, gerenciamento centralizado dos usuários e uma administração mais eficiente das estações de trabalho corporativas.
